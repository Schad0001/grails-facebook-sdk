The Facebook SDK Plugin allows your Grails application to use the Facebook Platform and develop Facebook apps on Facebook.com or on web sites (with Facebook Connect). It is a port of the official Facebook PHP SDK V3.1.1 to Grails 2.0.

(Quick Reference)

facebook:picture

Display user profile picture with a specific size.

<facebook:picture facebookId="${facebookContext.user.id}" />

Attributes

Required attribute is :

  • facebookId User or page id.

Optional attributes are :

  • elementClass HTML element class attribute value.
  • elementId HTML element id attribute value.
  • linkEnabled Render the image as clickable (link to facebook profile).
  • protocol Define protocol. Can be http to https (default to current protocol).
  • type Define image size. Can be large, small or square (default).
  • height Specify a custom height.
  • width Specify a custom width.
